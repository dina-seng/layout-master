# Layout Master Pro

A **React + Tailwind CSS** playground to visually experiment with **Flexbox** and **Grid layouts**.  
This project helps you learn, test, and generate real CSS code for your layouts in an interactive way.

---

## ðŸŒŸ Features

- **Dynamic Flexbox Layout**
  - Toggle between **row** and **column** directions
  - Control **justify-content** with real CSS mapping
- **Dynamic Grid Layout**
  - Preconfigured **3-column grid** with customizable gaps
- **Interactive UI**
  - Responsive **control panel** with buttons and dropdowns
  - Live **visual output** of items
- **Generated CSS**
  - Automatically outputs **real CSS code** for the current layout
  - Copy code with a single click
- **Clean Architecture**
  - Flex and Grid layouts are separated into **independent components**
  - VisualOutput component handles layout switching
  - State stores **real CSS**, Tailwind is only for rendering
